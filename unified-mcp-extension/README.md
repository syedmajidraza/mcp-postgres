# Syed MCP Extension

## Overview
The Syed MCP Extension is a unified tool for managing MCP servers. It allows users to install, start, stop, and configure MCP servers directly from the extension.

## How to Start the MCP Server

1. **Install the Extension**:
   - Open Visual Studio Code.
   - Go to the Extensions Marketplace.
   - Search for "Syed MCP Extension" and install it.

2. **Set Up the MCP Server**:
   - Ensure the MCP server is properly configured.
   - Update the registry URL in the extension settings if needed.

3. **Start the MCP Server**:
   - Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P` on macOS).
   - Search for `Syed MCP Extension: Start Server` and select it.

4. **Stop the MCP Server**:
   - Open the Command Palette.
   - Search for `Syed MCP Extension: Stop Server` and select it.

5. **View Logs**:
   - Use the `Syed MCP Extension: View Server Logs` command to check server activity.

## Features
- Install MCP servers from the registry.
- Start, stop, and restart servers.
- Configure server settings.
- View server logs.

## Requirements
- Node.js and npm installed.
- MCP server configured and accessible.
