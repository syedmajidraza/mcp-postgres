# Documentation Assets

This directory contains screenshots and other documentation assets for the PostgreSQL Chatbot project.

## Screenshots

### screenshot-chatbot.png
Main chatbot interface showing:
- Connection status
- Quick action buttons
- Query results table
- Natural language input field

To update the screenshot:
1. Open http://localhost:9000 in your browser
2. Take a screenshot of the chatbot interface
3. Save it as `screenshot-chatbot.png` in this directory
